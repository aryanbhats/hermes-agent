## Root cause (so we don't re-debug this)

The project was originally at `~/Documents/0DevProjects/hermes-agent`, then moved into `infra/`. The existing `.venv` was created at the *old* path, so all 70 console-script shebangs (`hermes`, `hermes-acp`, `hermes-agent`, `pip`, etc.) hardcoded `#!/Users/aryanbhatia/Documents/0DevProjects/hermes-agent/.venv/bin/python3` — a path that no longer exists. Sourcing `activate` did nothing because PATH-resolved scripts then failed at the shebang. Aliases in `~/.zshrc` ALSO pointed at the old path. Two layers of staleness.
